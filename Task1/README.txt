Hua Rong Dao sliding puzzle game
Hua Rong Dao is a sliding puzzle that is popular in China. Check out the following page for some background story on the puzzle and an English description of the rules. http://chinesepuzzles.org/huarong-pass-sliding-block-puzzle/

A* search was implemented. Two Heuristic Functions for A* Search were used. The first one is the Manhattan distance heuristic, the simplest admissible heuristic function for this problem. Next, another advanced heuristic function that is admissible but dominates the Manhattan distance heuristic was proposed in heuristics.pdf. 

To validate the script follow the instructions inside hrd_validate.py file
